How to Set Up on Your Local Machine or LAN

1 Save the HTML file

Save this entire web page as an HTML file on your computer. Right-click anywhere on the page and select "Save As..." or use Ctrl+S (Cmd+S on Mac). Save it as html-to-pdf.html.

2 Run a local web server (optional but recommended)

For best performance, especially if you need to access it from other devices on your LAN, run a simple web server:

Using Python:

python -m http.server 8000

Using Node.js (if you have it installed):

npx http-server -p 8000

Using PHP:

php -S 0.0.0.0:8000

3 Access the application

Open your web browser and navigate to:

http://localhost:8000/html-to-pdf.html

To access from other devices on your local network, use your computer's IP address:

http://[YOUR-IP-ADDRESS]:8000/html-to-pdf.html

Replace [YOUR-IP-ADDRESS] with your computer's local IP address (e.g., 192.168.1.100).

4 Direct file access (simplest method)

If you don't need to access it from other devices, you can simply open the HTML file directly in your browser by double-clicking it.
